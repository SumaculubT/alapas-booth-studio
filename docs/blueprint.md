# **App Name**: SnapStrip Studio

## Core Features:

- Template Upload: Allow users to upload custom PNG overlay templates.
- Photo Capture: Enable capturing multiple photos using the in-app camera.
- Composite Generation: Automatically apply the selected template to the captured photos to generate a composite photo strip.
- Saving/Sharing: Enable saving the final image to the device and sharing it on social media platforms.

## Style Guidelines:

- Background color: Dark gray (#333333) for a modern, minimalist feel.
- Primary color: Teal (#008080) as the accent color to provide contrast and highlight key elements.
- Secondary color: White (#FFFFFF) for text and primary UI elements to ensure readability and a clean aesthetic.
- Body and headline font: 'Inter', a sans-serif, to maintain a clean, modern look.
- Use simple, minimalist icons in white to maintain a consistent and modern look.
- Employ a clean, grid-based layout with ample spacing to ensure a clutter-free user interface.
- Use subtle transitions and animations to provide feedback and enhance user engagement.
